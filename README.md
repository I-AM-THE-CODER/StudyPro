# Exam & Study Tracker

## Overview
The Exam & Study Tracker is a client-side web application designed to help students manage their exams, organize study materials, and track their study progress. Built with HTML, CSS, and JavaScript, this application provides a user-friendly interface for effective study management.

## Features
- **Dashboard**: The main entry point that provides an overview of exams, tasks, and study resources.
- **Calendar View**: A dedicated page for managing exam schedules and study sessions.
- **Study Tools**: Tools for generating flashcards and practice questions to enhance study efficiency.
- **Settings**: Options for customizing app preferences, including a dark mode feature.

## Getting Started
1. **Clone the Repository**: 
   ```
   git clone <repository-url>
   ```
2. **Open the Project**: Open the `index.html` file in your web browser to start using the application.
3. **Customize Styles**: Modify the CSS files in the `css` directory to change the look and feel of the application.
